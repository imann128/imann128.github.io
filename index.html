TSAuditor a time-series integrity & data-quality auditing framework

A lightweight, pure-Python library — pip install tsauditor — that scans a DataFrame and returns a structured report of structural problems, anomalies, and its core contribution: data-leakage detection, catching features that secretly reproduce the target before they ever reach a model. It grew out of a real bug in my OGDC equity model, where a same-day ChangeP feature inflated accuracy to 99.68%; once the leak was removed, accuracy fell to a realistic ~70%. TSAuditor exists so that class of mistake gets caught automatically.

Leakage detection (AUC-based) Frequency & missing checks Stationarity (ADF) Anomaly & spike flags Rich + JSON reports
